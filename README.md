# 🌄 Parallax Scrolling Website

A visually stunning **Parallax Scrolling Website** built using **HTML5, CSS3, and JavaScript**. This project creates an immersive browsing experience by implementing smooth parallax effects, layered backgrounds, and modern web design techniques.

- Smooth Parallax Scrolling Effect

- Modern and Attractive UI

- Cross-Browser Compatibility
